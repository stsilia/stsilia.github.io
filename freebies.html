ArXiv Scraper

Reading literature is vital, but staying up to date is not always easy - especially while balancing work and deadlines. This Python script scours the last 5 mailings from astro-ph for papers that contain specific keywords in their title. Before running, just open it in any editor to add the desired keywords (the default one is "exoplanet").
